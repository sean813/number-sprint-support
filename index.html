Number Sprint Support

Need help with Number Sprint?

Number Sprint is a fast arcade math game with Solo mode, Game Center multiplayer, Game Center leaderboards, ads, and an optional Remove Ads purchase.

Game Center Multiplayer

Multiplayer requires Apple Game Center. Make sure Game Center is enabled on your device and that you are signed in.

Leaderboards

Leaderboards use Apple Game Center. Scores may take time to appear or refresh.

Remove Ads

If you purchased Remove Ads and ads still appear, tap Restore Purchases on the Home screen.
